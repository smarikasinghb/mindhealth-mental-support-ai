# MindHealth — Digital Mental Health & Psychological Support System
A comprehensive AI-powered mental health platform for students in higher education, built for **Smart India Hackathon 2025** (Problem Statement SIH25092, Theme: HealthTech). Team CodeStorm — MITS/SIH/008. Advanced through **2 national rounds**.

## Overview
Mental health issues among students are widespread but severely underserved due to stigma, lack of access, and poor awareness. MindHealth addresses this with an integrated platform combining AI companionship, mood tracking, journaling, screen time monitoring, gamification, and community support — all in one judgment-free space.

## Features
- **AI Companion** — NLP-based conversational support for instant emotional insights
- **Mood Tracker** — AI analysis of daily mood entries to detect warning signs early
- **Journaling** — Sentiment-analyzed journal entries with personalized feedback
- **Screen Time Monitoring** — Real-time pattern detection for FOMO, app-switching, and isolation
- **Gamification** — Coins, streaks, badges, and unlockable themes to normalize self-care
- **Community Chat + Share Life Journey** — Anonymous peer support and safe discussions
- **Time Management & Student Tools** — Productivity features for academic stress
- **Counselling Centre Integration** — Bridges self-help and professional support
- **Motivating & Learning Resources** — Curated educational content for mental wellness

## Tech Stack
- **Frontend:** React.js (web), React Native (mobile)
- **Backend:** Node.js with Express, Python (Flask/FastAPI)
- **Database:** MongoDB / MySQL
- **AI/NLP:** Hugging Face Transformers, BERT/GPT, spaCy
- **Cloud & DevOps:** AWS / Azure / GCP, Docker, Kubernetes
- **Security:** OAuth2, SSL/TLS, HIPAA/GDPR compliance, end-to-end encryption
